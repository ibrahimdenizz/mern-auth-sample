# mern-auth-sample
## How to install
<p>Fork and clone repository</p>
<pre>git clone https://github.com/ibrahimdenizz/mern-auth-sample</pre>
<p>Install dependencies</p>
<pre>
npm install
cd client/ && npm install
</pre>

## Run the app
<p>For server</p>
<pre>npm run server</pre>
<p>For client</p>
<pre>npm run client</pre>
